A propos du Projet Uruk

Le projet Uruk est une communauté de personnes très motivés qui partagent un fort intérêt pour la culture libre et une société libre.Nous nous entraidons et partageons un objectif commun: celui de soutenir et créer des projets libres.Le projet uruk fournit une liste de projets que nos membres suivent et qui ont pour but de renforcer est de promouvoir la culture et la philosophie du logiciel libre (libre comme dans liberté!).
Le nom d'Uruk a été choisi en l'honneur d'Uruk (warka)une ville en Irak (En Savoir Plus)
Pour démontrer comment le mot "uruk" devrait être prononcé, nous avons inclus un guide audio:



Le logo symbolise la Palme Sumérienne avec neuf frondes, entourée par les rivières du Tigre et de l'Euphrate.

Le but de notre projet est :

  • D’établir des projets libres pour soutenir la communauté des logiciels libres.
  • Faire une distribution GNU / Linux libre Pour répondre aux besoins des utilisateurs.
  • Apporter un soutien aux langues moins courantes dans la communauté du logiciel libre.
  • Répandre la philosophie du logiciel libre dans les endroits où la philosophie du logiciel libre est peu présente voir absente.
  • Se focaliser sur l'accessibilité de tous les utilisateurs et en particulier d'aider les personnes souffrant d'un handicap pour leur permettre d'utiliser un ordinateur le plus simplement possible.




Projets

Voici la liste suivante des sous-projets établie par notre équipe.

.
Uruk GNU/Linux Uruk GNU / Linux est un système d'exploitation libre pour la maison,la famille, les petites entreprises et les centres éducatifs basés sur Trisquel.
Uruk Cleaner Uruk Cleaner Uruk Cleaner est un programme pour nettoyer votre système de fichiers ainsi que les journaux système.
UPMS UPMS (Uruk Package Manager Simulator) est un simulateur de gestionnaire de paquets. Ce programme permet de simuler les commandes populaires d'un gestionnaire de paquets . Cela signifie que vous pouvez installer, désinstaller, mettre à jour et supprimer des paquets avec n’importe quel gestionnaire de paquets de votre choix.
Uruk Ocr Server Un petit serveur web simple et puissant, OCR, son utilisation est de convertir des Images en texte.
Irc log recorder L’enregisteur de journaux Irc est un enregistreur de journaux irc , un bot de regarder tout ce qui se passe dans un canal irc et d’enregistrer le journal . Irc log recorder possède un serveur Web Pour permettre aux utilisateurs d'accéder aux journaux à partir d’un navigateur Web et telnet.
Masalla Icon Theme Theme d’Icones Masalla thème d'icônes pour * NIX OS inspiré par la dernière tendance de design plat.
Uruk Cloud IDE Le projet Devbox est libre comme dans Liberté , Multiplateforme, avec un environnement de développement intégré mais pas uniquement limité au logiciel open source OpenStack .Ce logiciel est fortement recommandé pour les instituts, les sociétés de développement de logiciels et les développeurs. Le projet est encore en version bêta.
Le Site web du projet Uruk Le Code source du projet , le site web et Uruk GNU/Linux est basé sur un Processus collaboratif pair à pair source code
Rose Crypt Promouvoir et faciliter le chiffrement et déchiffrement des fichiers en utilsant AES.
Tabannusi c'est un systéme de construction de dépendances git qui surveille le dépot git et en cas de modification du dépot il construit le code qui existe dans le dépot et il l'envoie sour forme de rapport de construction à travers l'email
quicpass quicpass c'est l'administrateur et le générateur des mots de passe à travers la ligne de commande en utilisant les services de UNIX.
