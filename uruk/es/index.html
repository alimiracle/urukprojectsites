Acerca del Proyecto Uruk

El proyecto Uruk es una comunidad altamente motivada de personas compartiendo un fuerte interés en la cultura libre y sociedad libre. Todos nosotros nos ayudamos y compartimos el propósito en común de ayudar y construir proyectos libres. El proyecto Uruk provee una lista de metas que nuestros miembros siguen con el fin de promover y fortalecer el software y cultura libre.
El nombre Uruk fue elegido en honor de la ciudad Uruk (Warka) en Iraq (Leer más)
Para indicar como la palabra "uruk" se debe pronunciar, nosotros incluimos un archivo de audio:



El logo es un símbolo de la Palma Sumeria con nueve hojas, rodeada por los ríos Tigris y Éufrates.

La meta de nuestro proyecto es:

  • Establecer y mantener proyectos libres para ayudar a la comunidad de software libre..
  • Desarrollar una distribución libre GNU/Linux para satisface las necesidades de los usuarios.
  • Dar soporte a los idiomas menos comunes para la comunidad de software libre.
  • Difundir la filosofía del software libre en lugares donde esta filosofía es débil.
  • Enfocarse en la accesibilidad para ayudar a todos los usuarios, y en particular para quienes tiene discapacidades, operar sus cumputadores de la forma má fácil posible.




Proyectos

La siguiente lista contiene sub-proyectos de nuestro equipo.

Uruk GNU/Linux Uruk GNU/Linux es un sistema operativo completamente libre para usuarios domésticos, pequeñas empresas y centros educativos basado en Trisquel.
Uruk Cleaner Uruk Cleaner es un programa que limpia tu sistema desde los archivos cache y logs.
UPMS UPMS (Uruk Package Managers Simulator, Simulador de gestor de paquetes de Uruk). Este programa puede simular varios comandos comunes de gestores de paquetes, lo que significa que tu puedes instalar, desisntalar, actualizar y remover los paquetes usando cualquier gestor de paquetes que prefieras usar.
Servidor Uruk Ocr Un simple, pequeño, poderoso servidor web OCR usado para convertir imágenes en un texto.
Registrador Irc log El registrador Irc log es un bot irc que registra toda la comunicación de un canal IRC. Está equipado por un servidor web para hacer que los usuarios tengan acceso a los registros desde un navegador de internet y/o telnet.
Tema de ícono Masalla Tema de ícono para el SO *NIX inspirado por el moderno diseño de tendencia plano.
Uruk Cloud IDE El Proyecto UruCloudIDE es un proyecto libre multiplataforma de entorno de desarrollo integrado que se ejecuta principalmente, pero no exclusivamente, en una nube de software libre y de código abierto. Este software es altamente recomendado para instituciones, compañías de desarrollo de software y desarrolladores. El proyecto todavía está en beta.
La web de Uruk El código fuente del proyecto Uruk y la web de Uruk GNU/Linux, basada en el código fuente de la web Peers
Rose Crypt Promover y facilitar el cifrado y descifrado de los archivos usando un algoritmo AES
Tabannusi Tabannusi es un sistema de compilación continuo, que está viendo el repositorio git , si tú haces una nueva encomienda al repositorio, la aplicación va a compilar el código y enviarte por correo el registro de la compilación.
quicpass quickpass is un gestor de contraseñas de interfaz CLI y un generador de contraseñas al azar usando solamente las utilidades comúnmente disponibles en Unix.



Contacto

Usted puede contactarnos a través de uno de los siguientes métodos:

  • Únase a nuestra lista de correo. Para suscribirse a la lista, sólo envía un correo a uruk-request@lists.tuxfamily.org con el asunto subscribe. Un correo se reenviará para confirmar la petición. Para cancelar la suscripción sigue el mismo procedimiento, pero con el asunto unsubscribe. Y para seguir siendo fieles a nuestro status GNUisance, es necesaria la confirmación también en este caso. Para postear en la lista de correo envía tu mensaje a uruk@lists.tuxfamily.org

  • Chatea con nosotros en irc.freenode.net en nuestro canal #uruk-project
  • O póngase en contacto con los principales los administradores del proyecto por correo:
